# mini-recsys
mini recommender system with FastAPI (WIP)

## Environments
- Python 3.9.4

```bash
pip3 install -r requirements.txt
```

## Run

```bash
export PYTHONPATH=src/. 
python3 src/app/main.py
```

## URL
- http://localhost:5000/
- http://localhost:5000/docs
